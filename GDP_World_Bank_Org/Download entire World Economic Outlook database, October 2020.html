Have a WEO Data Question? Submit it through the WEO Question Form.

The entire World Economic Outlook database is available for download in Tab Delimited Values format and SDMX format. All available data that can be released to the public are incorporated into both formats.

Contents of the downloads

The Tab Delimited Values downloads below consist of two files. The "By Countries" file contains series data for all countries, and the "By Country Groups" file contains series data for all country groups (aggregated data). As many series are either not available or not applicable, there will be empty cells in the dataset. These files are compatible with most commercial statistical packages and Microsoft Excel.

The SDMX downloads below consist of two files. The SDMX Data file contains a data file with series for all countries and country groups and the associated schema file. The SDMX Data Structure Definition file contains the associated data structure definition.

Database version

By default, results are returned from the most recent WEO report dataset. Occasionally some countries' data are adjusted, typically when index years or currencies change.
